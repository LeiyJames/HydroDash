# HydroDash | The Modern Water Refill Business App

HydroDash is a premium, all-in-one digital ecosystem designed specifically for water refilling stores. It bridges the gap between store owners, delivery riders, and customers through a unified, high-performance platform.

## 🚀 Vision
To transform the "chaos" of manual water delivery management into a streamlined, digital-first experience that protects revenue and builds customer trust.

## 🛠️ Tech Stack
*   **Framework**: [Nuxt 3](https://nuxt.com/) (Vue.js 3 Meta-framework)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **UI Components**: [Shadcn Vue](https://www.shadcn-vue.com/)
*   **Icons**: [Lucide Vue Next](https://lucide.dev/)
*   **State Management**: Pinia
*   **Animation**: Tailwind animate & custom transitions

---

## 📱 Role-Based Ecosystem
*   **Admins**: Full visibility on revenue, rider status, and order history.
*   **Riders**: Optimized delivery lists, distance-based sorting, and digital remittance markers.
*   **Customers**: 1-tap reordering, real-time status tracking, and loyalty point rewards.

## 🛠️ Getting Started for Developers

### Prerequisites
* Node.js (Latest LTS)
* npm

### Installation
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

The application will be available at `http://localhost:3000`.
